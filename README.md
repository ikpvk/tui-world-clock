# TUI World Clock

A terminal-based multi-timezone clock built with Python and Textual.

## Features

- Multi-timezone support
- Searchable timezone picker
- Arrow key navigation
- 12h / 24h toggle
- Persistent config storage
- Cross-platform config handling (platformdirs)

## Controls

- `a` → Add timezone
- `d` → Delete selected timezone
- `t` → Toggle 12h / 24h format
- `q` → Quit
- Arrow keys → Navigate

## Installation

```bash
pip install -r requirements.txt
python app.py
